Directive BB.Directives:ngOptions

there is a some sort of 'redraw' bug in IE with select menus which display more than one option and the options are dynamically inserted. So only some of the text is displayed in the option until the select element recieves focus, at which point all the text is disaplyed. comment out the focus() calls below to see what happens.

  • link(scope, element, attrs)

directive in module BB

Description

Directive Breadcrumbs
Used to load the application's content. It uses ng-include.

loadStep        = $scope.loadStep
$scope.steps    = $scope.bb.steps
$scope.allSteps = $scope.bb.allSteps

Has the following set of methods:

  • $scope.loadStep(number)
  • lastStep()
  • currentStep(step)
  • atDisablePoint()
  • $scope.isDisabledStep(step)

Usage

as attribute
<ANY breadcrumbs
     $scope="{service}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
$scopeservice

Scope is an object that refers to the application mode.

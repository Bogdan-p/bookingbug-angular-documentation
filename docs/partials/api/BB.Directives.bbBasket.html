Directive BB.Directives:bbBasket

restrict: 'A'
replace: true
scope : true
templateUrl : (element, attrs) ->
if _.has attrs, 'mini'
then PathSvc.directivePartial "_basket_mini"
else PathSvc.directivePartial "basket"
controllerAs : 'BasketCtrl'

Has the following set of methods:

  • templateUrl(element, attrs)
  • controller($scope, $modal, BasketService)
  • link : (scope, element, attrs)
    Stop the default action of links inside directive. You can pass the $event object in from the view to the function bound to ng-click but this keeps the markup tidier.

BBModel
object in module BB

Description

Build a dynamic injector for each of the models. This creates a service that is capable of creating any given model. It uses dynamic injection, to avoid a cuicular dependancy - as any model needs to be able to create instances of other models

Usage

BBModel($q, $injector);

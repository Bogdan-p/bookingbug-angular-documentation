popover
directive in module BB

Description

Directive BB.Directives:popover

The default behavior for the 'click' event for the popover is to hide the popover if the click occurs on the element but it doesn't hide if you click anywhere else as the event has to occur on the element. So we add additional check on the container to check if the popover is still open on a click event. Clearly this will fail if the event propogation is stopped in the code, but that is bad practise so shouldn't be happening.

Usage

as element:
<popover>
</popover>
as attribute
<ANY popover>
   ...
</ANY>

Directive info

  • This directive executes at priority level .

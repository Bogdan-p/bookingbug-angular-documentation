bbLocalNumber
directive in module BB

Description

Directive BB.Directives:bbLocalNumber
Adds a formatter that prepends the model value with zero. This is useful for nicely formatting numbers where the prefix has been stripped, i.e. '7875123456'

Has the following set of methods:

  • link(scope, element, attr, ctrl)

Usage

as attribute
<ANY bb-local-number>
   ...
</ANY>
